#InputRegex

input regex를 모은 enum 

#InputView

### readBridgeSize

    다리의 길이를 입력받음

### validateBridgeSize

    입력 받은 다리의 길이가 올바른지 검증

### readMoving

    사용자가 움직일 방향을 입력받음

### validateMoving

    입력 받은 방향이 U,D 인지 검증

### readGameCommand

    사용자의 게임 재진행 여부 를 입력 받음

### validateGameCommand

    입력한 게임 진행 여부가 R,Q 인지 검증
